#    Создать сайт мастерской по ремонту бытовой техники.    #
#############################################################

------------------------------------------------------------------------------------------
    Сайт должен включать:
------------------------------------------------------------------------------------------
        1. Регистрацию клиентов
        2. Прием заявок от клиентов
        3. Контакты и ссылки на мессенджеры администратора
        4. Сервис по отслеживанию заказов и обратную связь с клиентом
        5. Прайс - лист с разделением на услуги и магазин с автоматическим обновлением
        6. Корзина для заказов

------------------------------------------------------------------------------------------
    Базы данных:
------------------------------------------------------------------------------------------
        1. База клиентов
        2. База мастеров и работников
        3. База заявок с разделением на "выездные" и "сервисные"
        4. База для учета финансов
        5. База по заказам

------------------------------------------------------------------------------------------
    Описакние модели клиентов:
------------------------------------------------------------------------------------------
        1. Имя клиента
        2. Телефон клиента
        3. Электронная почта клиента
        4. Персональная скидка для клиента

------------------------------------------------------------------------------------------
    Описание модели мастеров и работников:
------------------------------------------------------------------------------------------
        1. Имя и фамилия сотрудника
        2. Телефон сотрудника (возможно несколько телефонов)
        3. Процентная ставка
        4. Штрафы и поощрения -> Вынесу в бизнес логику
        5. Род занятия (специализация) -> Позже добавлю
        6. Привязка к выполняемым заказам

------------------------------------------------------------------------------------------
    Описание модели заявки на ремонт:
------------------------------------------------------------------------------------------
        1. Название заявки
        2. Категория техники -> заменил на "Модель техники"
        3. Краткая информация о неисправности
        4. Принадлежность клиенту
        5. Статус заявки ("готова", "принята", "в работе", "рекламация")
        6. Информация по ремонту, согласование с клиентом стоимости ремонта
        7. Привязку к мастеру
        8. Вид проведения работ ("выездная", "сервисная")
        9. Стоимость ремонта
        10. Дата принятия завки и срок выполнения
        11. Фото (при необходимости)

------------------------------------------------------------------------------------------
    Опсиание модели заказа:
------------------------------------------------------------------------------------------
        1. Наименование
        2. Статус заказа ("доставка", "доставлено")
        3. Стоимость заказа
        4. Привязка к клиенту

------------------------------------------------------------------------------------------
    Описание модели товара:
------------------------------------------------------------------------------------------
        1. Наименование товара
        2. Стоимость товара
        3. Описание харакеристик
